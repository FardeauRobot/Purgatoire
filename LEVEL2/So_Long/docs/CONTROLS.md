# Controls

Default keybinds (adjust to your implementation):
- `W` / `↑` — move up
- `A` / `←` — move left
- `S` / `↓` — move down
- `D` / `→` — move right
- `Esc` or window close — quit

Common UI behavior: print move count to stdout after each move; exit when reaching `E` with all `C` collected.

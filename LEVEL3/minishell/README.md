# minishell
## Description
The First project of 42 with partner, we need to write a small
version of shell, the famous, the uniq bash.

For this project we need to implement a small version of bash, with the following features:
- Parsing of the command line
- Execution of commands
- Handling of built-in commands
- Handling of redirections
- Handling of pipes
- Handling of signals

## Instructions
`make` to compile the project, then run

`./minishell`
